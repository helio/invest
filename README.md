# ENV Variables etc.
You can configure a lot of stuff through ENV-Variables.

```bash
JWT_SECRET='random secret for jwt etc.'
ZAPIER_SECRET='random secret for zapier hooks'
SITE_ENV='DEV|STAGE|PROD|TEST'
DB_USERNAME=
DB_NAME=
DB_HOST=
DB_PASSWORD=
```
